# 🛒 Lista de Supermercado com Soma Automática

Um aplicativo simples, prático e interativo desenvolvido para ajudar no controle de gastos durante as compras. Com ele, você pode listar os produtos que precisa comprar, registrar o preço de cada um e ver o valor total da compra atualizado em tempo real.

## 🚀 Funcionalidades

* **Cadastro de Itens:** Adicione o nome do produto à sua lista de compras.
* **Preço Individual:** Insira o valor unitário ou total de cada produto inserido.
* **Soma Automatizada:** O aplicativo calcula e exibe instantaneamente o valor total de todos os itens cadastrados.
* **Interface Dinâmica:** Atualização visual imediata assim que um novo item é adicionado ou removido.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias web fundamentais:

* **HTML5:** Para a estruturação dos elementos da página e formulários de entrada.
* **JavaScript (ES6):** Para a lógica de manipulação do DOM, cálculo dos valores e dinamismo da lista.
* **CSS3:** (Opcional, adicione se tiver) Para a estilização e responsividade do layout.

## 💻 Como Executar o Projeto

1. Faça o clone deste repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` diretamente em qualquer navegador web (Chrome, Firefox, Edge, etc.).
3. Comece a montar a sua lista de compras!
